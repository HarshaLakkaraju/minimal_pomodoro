# How to Use Pomodoro Timer

## Basic Usage
1. The timer starts at 45 minutes
2. Press SPACE to start/pause
3. Work until timer completes
4. Take a 5-minute break
5. Repeat!

## Keyboard Shortcuts
- `Space` - Start/Pause
- `R` - Reset timer  
- `M` - Show/hide menu
- `C` - Change duration (25/45/60/90 min)
- `S` - Stop alarm sound
